# 个人工具箱

一个个人使用的小网站，用来聚合常用轻量工具。当前是纯静态页面，无需安装依赖，直接用浏览器打开即可。

## 已有工具

- 总览：首页展示工具入口和关键状态
- 每日花销：记录日期、分类、金额和备注，自动汇总今日与本月支出
- 随手便签：保存临时文字，内容保存在当前浏览器
- 快捷入口：放常用网页、本地服务或项目入口

## 使用

直接打开 `index.html` 即可使用。记账和便签数据都保存在浏览器本地；清除浏览器站点数据后会一并删除，重要数据建议定期导出备份。

## 后续可扩展

可以继续加入换算器、待办清单、日期计算器、文本处理器、常用链接管理等小工具。

## 部署

项目已经配置 GitHub Pages 自动部署。把代码推送到 `main` 分支后，GitHub Actions 会自动发布静态网站。

首次使用时，在 GitHub 仓库的 `Settings -> Pages` 中确认部署来源为 `GitHub Actions`。之后每次推送 `main` 都会自动更新网站。
