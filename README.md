# 个人工具箱

一个个人使用的小网站，用来聚合常用轻量工具。当前是纯静态页面，无需安装依赖，直接用浏览器打开即可。

## 已有工具

- 总览：首页展示工具入口和关键状态
- 每日花销：记录日期、分类、金额和备注，自动汇总今日与当前发薪周期支出，并提供分类占比、日均和趋势分析
- 随手便签：保存临时文字，可选择同步到云服务器
- 日期计算：计算日期间隔，也可按天数推算日期
- 文本处理：统计字符、行数和词/字，支持清理空格、删除空行、复制结果
- 快捷入口：放常用网页、本地服务或项目入口
- 云同步：通过你的服务器 API 保存到 SQLite

## 使用

直接打开 `index.html` 即可使用。默认使用浏览器本地保存；在“云同步”里填入 API 地址、访问令牌和每月发薪日后，记账和便签会同步到你的服务器 SQLite，统计会按发薪日到下个发薪日前一天计算。

## 后续可扩展

可以继续加入换算器、待办清单、常用链接管理、密码生成器、番茄钟等小工具。

## 部署

项目已经配置 GitHub Pages 自动部署。把代码推送到 `main` 分支后，GitHub Actions 会自动发布静态网站。

首次使用时，在 GitHub 仓库的 `Settings -> Pages` 中确认部署来源为 `GitHub Actions`。之后每次推送 `main` 都会自动更新网站。
